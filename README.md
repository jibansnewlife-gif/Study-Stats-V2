# StudyStats Native

A native desktop version of StudyStats built using **PySide6 (Qt for Python)**.

## 🚀 Features

* Native desktop UI (no browser)
* Sidebar navigation
* Modular architecture (UI + Core separation)
* Custom app icon
* QSS-based styling system

## 📂 Project Structure

```
myapp/
│
├── main.py
├── ui/
├── core/
├── styles/
├── assets/
├── data/
```

## ▶️ How to Run

1. Install dependencies:

```
pip install PySide6
```

2. Run the app:

```
python main.py
```
